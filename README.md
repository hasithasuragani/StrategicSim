# StrategicSim
# Smart Strategy Simulator
An AI-powered tool that simulates different strategies based on user scenarios.

# Project Description
This project uses AI to analyze a given situation and generate possible strategies.
Users can input a scenario and simulate different outcomes. 
The system generates strategic suggestions and displays the results interactively.

# Features:
- Accepts user scenarios as input
- Generates AI-based strategies
- Simulates possible outcomes
- Displays strategy results dynamically
- Uses Google AI API for intelligent responses

# Technologies Used:
- HTML
- Type Script
- CSS
- JavaScript
- Google AI API
- Google AI Studio

# Steps to Run the Project:
1. Download or clone the repository.
2. Open the project folder.
3. Add your Google AI API key in the configuration file.
4. Run the project using a local server or open the index.html file.
5. Enter a scenario and click "Simulate Output".

# API Setup:

This project requires a Google AI API key.

Steps:
1. Go to Google AI Studio.
2. Generate an API key.
3. Add the API key in the config file or environment variable.

Without the API key, the system will show mock data instead of real AI-generated results.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
